## Installation

Install Agency Swarm using pip:
<CodeBlock>
```bash
pip install agency-swarm
```
</CodeBlock>


## Getting Started

When it comes to getting started with Agency Swarm, you have three options:

<Steps>
  <Step title="From Scratch">
    This is the best option if you want to get a feel for the framework and understand how it works. You can start by creating your own agents and tools, and then use them to create your own agencies. [Learn more](/quickstart/from-scratch)
  </Step>
  <Step title="Use Genesis Agency">
    This is the best option if you want to get started quickly and don't want to spend time creating your own agents and tools. You can use the Genesis Agency to create your agent templates and tools, and then fine-tune them to your needs. [Learn more](/quickstart/genesis-agency)
  </Step>
  <Step title="Cursor">
    Use Cursor to create your own agents and tools. [Learn more](/quickstart/cursor-ai)
  </Step>
</Steps>
